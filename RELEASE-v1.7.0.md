# Clipo v1.7.0 — Speak Your Language

**Your clipboard, your language. 10 languages, zero friction.**

---

## What's New

### 🌍 Multi-Language Support
Clipo now speaks **10 languages** out of the box. Switch instantly from Settings → General → Language with no restart required.

| Language | Code |
|---|---|
| English | en |
| 简体中文 | zh-Hans |
| 繁體中文 | zh-Hant |
| 日本語 | ja |
| 한국어 | ko |
| Français | fr |
| Deutsch | de |
| Español | es |
| Русский | ru |
| Português | pt |

Every UI surface is localized — the panel, settings, permission window, menu bar, toast notifications, context menus, and even the onboarding hints.

### 🔧 UI Polish
- **Larger settings icon** in the Clipo panel for easier targeting.
- Type filter pills, footer shortcuts, and hover actions all localized.

## Install

1. Download `Clipo.dmg`.
2. Drag **Clipo** into your **Applications** folder.
3. Launch and grant **Accessibility** permission when prompted.
