# Clipo

Save text to 9 clipboard slots with hotkeys. Browse history. Paste instantly. macOS 13+.

## Install

1. Download `Clipo.dmg` from [Releases](https://github.com/hanazargames/clipo/releases).
2. Drag into **Applications**.
3. Launch and grant **Accessibility** permission.

## Use

| Action | Shortcut |
|---|---|
| Open panel | `⌥Space` |
| Save selection to Slot 1–9 | `⌥⌘1` … `⌥⌘9` |
| Copy Slot 1–9 to clipboard | `⌥1` … `⌥9` |

In the panel: **↑↓** navigate, **↵** paste, **⌘↵** copy, **⌘P** pin, **⌫** delete. Type to search.

## Settings

Open from the menu bar or press `,` inside the panel.
